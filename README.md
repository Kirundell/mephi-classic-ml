# Прогнозирование эффективности химических соединений против вируса гриппа

## Курсовая работа по классическому машинному обучению. 
Построение моделей для предсказания эффективности (IC50), цитотоксичности (CC50) и селективности (SI) химических соединений.

## :file_folder: Структура

```
├── models                                 # Папка с весами моделей
├── eda.ipynb                              # Разведочный анализ данных
├── regression_ic50.ipynb                  # Регрессия IC50
├── regression_cc50.ipynb                  # Регрессия CC50
├── regression_si.ipynb                    # Регрессия SI
├── classification_ic50_more_median.ipynb  # Классификация IC50 > медианы
├── classification_cc50_more_median.ipynb  # Классификация CC50 > медианы
├── classification_si_more_median.ipynb    # Классификация SI > медианы
├── classification_si_more_8.ipynb         # Классификация SI > 8
├── report.pdf                             # Аналитический отчёт
├── data.xlsx                              # Исходные данные
└── data_processed.csv                     # Обработанные данные
```
